# Rails Skills

A collection of Ruby on Rails skills for AI coding agents like [OpenCode](https://opencode.ai), Claude Code, and other compatible tools.

## Available Skills

| Skill | Description |
|-------|-------------|
| `minitest` | Expert guidance for writing fast, maintainable Minitest tests in Rails applications |

## Installation

Install skills using the `npx skills` CLI:

```bash
npx skills add ThinkOodle/rails-skills
```

This downloads the skills and configures them for use with your AI agent.

### Install a Specific Skill

```bash
npx skills add ThinkOodle/rails-skills --skill minitest
```
